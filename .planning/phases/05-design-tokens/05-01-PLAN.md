**Keyframe overrides (redefine inside the media query):**

1. `slide-in-from-right` — Remove translateX(100%), keep only opacity 0→1, duration ~200ms
2. `fade-out` — Already opacity-only, no change needed (but include for completeness if desired)
3. `modal-fade-in` — Already opacity-only, no change needed
4. `modal-fade-out` — Already opacity-only, no change needed
5. `modal-scale-up` — Remove scale(0.95) and translateY(8px), keep only opacity 0→1
6. `modal-scale-down` — Remove scale(0.95) and translateY(8px), keep only opacity 1→0
7. `fade-in-up` — Remove translateY(8px), keep only opacity 0→1
8. `widget-enter` — Remove translateY(12px), keep only opacity 0→1. Keep stagger delays unchanged (the `.stagger-*` classes stay as-is — stagger timing is not motion)
9. `chart-grow` — Remove scaleY(0.8), replace with opacity 0→1 fade (~200ms duration)

**Utility class overrides (inside the same media query):**

- `.animate-in` — animation uses redefined slide-in-from-right (fade-only), keep 0.3s or reduce to 0.2s
- `.slide-in-from-right-full` — same as above
- `.animate-fade-in-up` — animation uses redefined fade-in-up (fade-only)
- `.animate-modal-scale-up` — animation uses redefined modal-scale-up (fade-only)
- `.animate-modal-scale-down` — animation uses redefined modal-scale-down (fade-only)
- `.animate-widget-enter` — animation uses redefined widget-enter (fade-only)
- `.animate-chart-enter` — animation duration reduced to 0.2s, transform-origin removed (no scaleY), uses redefined chart-grow (fade-only)

**Implementation approach:** Use a single `@media (prefers-reduced-motion: reduce)` block. Redefine the keyframes inside it so that the same animation names play but without spatial transforms. This is the cleanest approach — components keep their existing class names, the CSS cascade handles the rest.

**Do NOT:**
- Remove or modify any existing keyframes/classes outside the media query
- Disable animations entirely (opacity transitions must still play)
- Remove stagger delay classes
- Touch shadow tokens (they are already correct)
